# Idea 6: Onboarding Pathfinder Agent

New hire's first day: they tag the agent in Slack with their team name. The agent uses Sourcegraph deepsearch to build a personalized onboarding path — not docs, but an actual guided tour through the codebase. It identifies the team's most-touched files (via commit_search), the key abstractions they own (via symbol navigation), the upstream/downstream dependencies (via find_references), and generates a 'start here, then here, then here' reading order with explanations at each stop. Updates itself as the codebase evolves. Sourcegraph's code intelligence replaces the 'shadow a senior dev for 2 weeks' pattern.
