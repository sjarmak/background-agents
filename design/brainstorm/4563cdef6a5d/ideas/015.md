# Idea 15: Cross-Repo Test Gap Analyzer

Agent analyzes a code change (CI trigger) and uses Sourcegraph find_references to identify all cross-repo code paths that exercise the changed code. Then checks whether those paths have test coverage by searching for test files that import/call those symbols. Identifies 'test gaps' — code paths in OTHER repos that depend on the changed behavior but have no tests covering the specific interaction. Posts: 'Your change to auth.validate() is called by 8 services. 3 of them have no tests exercising this code path. Here are the specific untested call sites.' The shape: testing intelligence derived from cross-repo dependency analysis.
