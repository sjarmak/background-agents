# Idea 22: Code Clone Divergence Tracker

Identifies code that was copy-pasted across repos (via Sourcegraph nls_search for semantically similar code blocks) and then DIVERGED over time. Uses diff_search to trace when the copies started evolving independently. Posts alerts: 'These 3 repos all forked the same auth middleware 8 months ago. Since then, repo-A fixed a security bug that repos B and C still have. Here are the specific divergent lines.' The shape: clone detection + temporal divergence analysis = finding bugs that exist in copies but were fixed in the original.
