# Idea 5: API Contract Fuzzer via Cross-Repo Usage Mining

Agent mines Sourcegraph for every actual call site of an API across all repos. Compares the set of actually-used parameter combinations against the API's declared interface to find: (1) parameters nobody uses (dead surface area), (2) implicit contracts (everyone passes null for field X), (3) undocumented invariants (all callers check error code Y before proceeding). Triggered on-demand from Slack when planning an API redesign. Sourcegraph's cross-repo find_references + read_file is the only way to mine actual usage patterns at scale without instrumenting production.
