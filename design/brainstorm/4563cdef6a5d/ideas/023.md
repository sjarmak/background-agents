# Idea 23: Ambient Codebase Awareness Feed

Not triggered by events or requests — runs continuously. Agent uses Sourcegraph to monitor a configurable set of symbols/files/patterns across all repos. When anything changes, it posts to a dedicated Slack channel like an RSS feed for code: 'auth.validate() gained a new caller in payments-v2 repo 2h ago', 'The deprecated UserV1 type was referenced in a new file', 'Error handling pattern in billing changed from try/catch to Result type'. Engineers subscribe to the symbols/patterns they care about. The shape: publish-subscribe model for code changes at the symbol level, not the file level. Sourcegraph's symbol-level cross-repo monitoring is the unique enabler.
