# Idea 29: Micro-Refactoring Nightshift

Agent operates during off-hours (cron trigger, e.g., 2am). Uses Sourcegraph nls_search to find small, safe, well-bounded refactoring opportunities across repos: renaming inconsistent variables to match conventions, extracting duplicated 3-line patterns into shared utilities, updating deprecated API calls to their modern equivalents. Each change is tiny (1-5 lines) and mechanically verifiable. Opens PRs that are trivially reviewable in the morning. The shape: autonomous overnight batch worker that makes many small improvements across many repos — not a single task executor but a continuous maintenance daemon.
