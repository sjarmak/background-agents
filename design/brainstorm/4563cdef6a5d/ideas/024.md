# Idea 24: Interview Prep Agent: Codebase Deep Dives for Hiring

When a hiring manager tags the agent before an interview, it generates a codebase-grounded technical interview question set. Uses Sourcegraph deepsearch to find the most interesting architectural decisions, tricky edge cases, and subtle design trade-offs in the team's code. Generates questions like 'We chose X pattern here instead of Y — what trade-offs do you see?' with actual code snippets. The interviewer gets domain-specific questions rooted in real code, not generic LeetCode. Shape: agent generates human-to-human interaction material from codebase analysis.
