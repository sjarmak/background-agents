# Idea 7: Dead Code Bounty Hunter

Scheduled weekly agent that uses Sourcegraph find_references across the ENTIRE org to identify symbols with zero cross-repo references — functions, types, endpoints that nothing calls anymore. But goes further: uses diff_search to find WHEN the last caller was removed, and commit_search to understand WHY. Posts a prioritized 'dead code bounty board' to Slack with estimated cleanup effort and risk assessment. Engineers claim bounties. This requires org-wide reference analysis — impossible with single-repo grep.
