# Idea 3: Cross-Repo Semantic Merge Conflict Predictor

Agent runs on every PR open. Uses Sourcegraph to find all OTHER open PRs across all repos that touch the same symbols (via find_references + go_to_definition). Predicts semantic merge conflicts BEFORE they happen — not git conflicts (textual overlap) but logical conflicts (two PRs changing the same API contract from different repos). Alerts both PR authors in Slack with the specific symbols at risk. This is impossible without cross-repo symbol-level intelligence.
