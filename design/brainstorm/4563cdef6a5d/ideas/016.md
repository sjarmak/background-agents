# Idea 16: Codebase Storyline Generator for Non-Engineers

Product managers, designers, or executives ask in Slack: 'How does the checkout flow work?' The agent uses Sourcegraph deepsearch + go_to_definition to trace the end-to-end code path, then translates it into a non-technical narrative with a sequence diagram. No code shown — just 'When a user clicks Buy, the frontend sends a request to the cart service, which validates inventory by calling the warehouse API, then...' Uses Sourcegraph's semantic navigation to follow the actual code path, not guess. Unique shape: the consumer is a non-engineer, the output is a narrative, the intelligence is code-graph traversal.
