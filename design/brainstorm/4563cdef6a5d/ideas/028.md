# Idea 28: Codebase Diff Debate: Two Agents Argue a Design Decision

When an architect posts a design question in Slack ('Should we use event sourcing or CQRS for the new billing system?'), TWO agents are spawned with opposing briefs. Each uses Sourcegraph deepsearch independently to find evidence in the actual codebase supporting their assigned position. Agent A argues for option 1 citing real code patterns, past failures, and existing infrastructure. Agent B argues for option 2 with different evidence from the same codebase. Posts a structured debate transcript to the Slack thread. Shape: adversarial multi-agent debate grounded in codebase evidence, not a single agent giving one recommendation.
