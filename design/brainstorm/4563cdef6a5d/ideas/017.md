# Idea 17: Regulatory Compliance Mapper

For regulated industries (fintech, healthcare). Agent uses Sourcegraph to maintain a live mapping between regulatory requirements and the code that implements them. When a regulation changes (input via Slack), agent uses nls_search + keyword_search to find all code implementing the affected requirement, then uses find_references to trace the full compliance surface. Posts: 'GDPR Article 17 (right to deletion) is implemented across 4 repos in 12 functions. Here are the specific code locations and their last audit dates.' The unique shape: the input is a legal/regulatory requirement, the output is a code-to-regulation traceability matrix.
